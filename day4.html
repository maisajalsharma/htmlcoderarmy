Relative Paths (for your own project files)

Relative paths are based on the location of your HTML file.

  • Same folder: If about.html is in the same folder as your main file: <a href="about.html">About</a> This opens about.html in the same folder.
  • Sub-folder: If your image is inside an images folder: <img src="images/logo.png" alt="Logo"> This loads logo.png from the images folder.
  • Parent folder: If the image is one folder above your HTML file: <img src="../logo.png" alt="Logo"> .. means â€œgo up one folderâ€.

Absolute Paths (external resources)

Absolute paths give the full web address (URL). Use them only for external websites/resources.

Example:

<a href="https://google.com">Google</a>
<img src="https://upload.wikimedia.org/.../html5.svg" alt="HTML5">

Rule: Use relative paths for your own project, absolute paths for external links.
